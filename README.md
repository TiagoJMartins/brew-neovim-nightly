# brew-neovim-nightly

Homebrew cask for neovim nightly

## Installation

```
brew tap TiagoJMartins/brew-neovim-nightly https://github.com/TiagoJMartins/brew-neovim-nightly.git
```
```
brew install --no-quarantine neovim-nightly
```
